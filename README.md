## About The Project


First team project, focused on working in synergy as a team with the Git tools.
User is given a choise which function to use:<br />
1st func : given a date (format: year month day) the output will be the number of the day out of 365 or 366 if its a Leap year.<br />
2nd func : given a date (format: year month) the output will be the number of days in a certain month with Leap year consideration.<br />
3rd func : given a date (format: year) the output will be whether the year is a Leap year or not.<br />
